Header

Try to scroll this page: the header hides on scroll-down and shows on scroll-up with CSS only.

This demo uses scroll-state(scrolled) container queries that respond to the most recent scroll direction.
Check it out in Chrome Canary with the experimental web platform features flag flipped on.

The demo is forked from the original by Fabrizio Calderan, which uses Scroll-Driven Animations.
